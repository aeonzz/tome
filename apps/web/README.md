# `apps/web`

The public web portal and landing page for the Tome project, built with Next.js.

## Features

- **Next.js App Router**: Optimized for speed and SEO.
- **Aesthetic UI**: Premium design using `@tome/ui`.
- **Type-safe API**: Seamless integration with `@tome/api`.

## Getting Started

### Scripts

- `pnpm dev`: Start the development server on port 3001.
- `pnpm build`: Build the production bundle.
- `pnpm start`: Start the production server.
- `pnpm lint`: Run ESLint to check for code issues.
- `pnpm format`: Format code using Prettier.
- `pnpm typecheck`: Check for TypeScript type errors.

## Tech Stack

- [Next.js](https://nextjs.org/)
- [Tailwind CSS 4](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)
